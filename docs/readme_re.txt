Original Retrieval For COD2 by Etromania - etromania@wclan.com - www.wclan.com
This is not a port of RE but a recreation Combining SD and CTF Gametypes
 
This version of Retrieval has been revised by =G4C=Dude4Him - dude4him@g4christ.com http://www.g4christ.com

Gametype Cvars - place these gametype options in your server config:

scr_re_roundlength "5"  //How long each round is (in minutes)
scr_re_graceperiod "15" //How long you can change weapons and join the game (in seconds)
scr_re_roundlimit  "0"  //Round limit - how many rounds are played
scr_re_scorelimit  "0"  //Score limit - score limit before map ends
scr_re_timelimit   "25" //Time limit for the entire match (in minutes)

The map rotation should appear as "gametype re map mp_breakout"

Changes in this version:

- Allies (instead of Axis) must retrieve the Axis Enigma Cipher Machine and return it to the extraction point (an open shipping crate at the grey square on the compass).
- The problem with the original version was if the Axis did not retrieve the Artillery map they still got a score. That is fixed in this version.
-New Objective models used.
-All Stock CoD2 maps now supported
-Custom map support added. See capatable maps below.

Special thanks to Raiders Game Net for the DTL tool. I couldn't make most of these maps work with RE without it.
